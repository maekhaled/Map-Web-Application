To add an object to the map, click on the location on the map where you would like to position your object. In the dialog box, select the type of object you would like to drop, enter the object description and click 'Add Object'.

To edit an existing object, select the object on the map. In the dialog box, edit the object's description and click 'Edit Object'.

To delete an existing object, select the object on the map. In the dialog box, click 'Delete Object' to confirm the deletion of the object.
